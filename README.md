# Neovim Translate
- A nvim plugin for translate
- version 0.0.1

## Requirement
- Python
- googletrans (PYPI package)
- neovim (PYPI package)


## Installing
- `pacman -S neovim python-neovim`

## Usage
- `:Translate`
		- translate the current line
